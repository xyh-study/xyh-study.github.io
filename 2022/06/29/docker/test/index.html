六 数据卷

6.1 数据卷的概念

  • 数据卷是宿主机中的一个目录或文件
  • 当容器目录和数据卷目录绑定后，对方的修改会立即同步
  • 一个数据卷可以被多个容器同时挂载
  • 一个容器也可以被挂载多个数据卷

九 docker compose

9.1 docker compose 服务编排

Docker Compose是一个编排多容器分布式部署的工具，提供命令集管理容器化应用的完整开发周期，包括服务构建,启动和停止。

  • 利用Dockerfile定义运行环境镜像
  • 使用docker-compose,ym定义组成应用的各服务
  • 运行docker-compose up启动应用
